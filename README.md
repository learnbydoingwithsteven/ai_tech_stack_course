# AI Technology Stack Course

A comprehensive educational platform for understanding the complete AI technology stack, from infrastructure to application layer.

## 🎯 Course Overview

This course covers the five critical layers of AI systems:

1. **Infrastructure Layer** - Hardware and deployment options
2. **Model Layer** - LLMs, SLMs, and model selection
3. **Data Layer** - Data sources, pipelines, and RAG
4. **Orchestration Layer** - Planning, execution, and review
5. **Application Layer** - Interfaces and integrations

## 🚀 Quick Start

### Running the Interactive App

```bash
# Backend
cd app/backend
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
python app.py

# Frontend (in another terminal)
cd app/frontend
# Open index.html in browser or use live server
```

## 🎓 Learning Path

1. Start with **Module 1: Introduction** to understand the big picture
2. Work through each layer sequentially
3. Complete hands-on exercises in action plans
4. Build the example drug discovery assistant
